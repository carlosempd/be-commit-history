## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

This project is the backend side for this frontend app ( [repo frontend](https://github.com/carlosempd/fe-commit-history) ). It is built with NestJs and it has a single endpoint `/commits` that lists all the commits for this repository or the frontend app repository, depending on the query param `repo` (this query param has two posible values, `be` or `fe` to select wich repo commits).

It is IMPORTANT for the frontend app [here](https://github.com/carlosempd/fe-commit-history) that this backend project is running in order to retrieve the commits and show them in the corresponding view list.


## Guide

First of all, you must check that you have NodeJS and NestJs installed. Visit [Nest](https://github.com/nestjs/nest) and [Node](https://nodejs.org/en/) to check the installation instructions. This step is optional since with the comand `npm install` you can install all the dependencies required to run the project.

Follow this steps to run the project:

1- Clone or Download this project. In a terminal run `git clone https://github.com/carlosempd/be-commit-history.git` to clone the repository using https protocol (Install [git](https://git-scm.com/downloads) in case you haven't installed it yet)

2- In the terminal navigate to the project folder. For example: `cd be-commit-history `. 

3- Install node modules using `npm install`. 

4 - Run the Development Server(`npm run start`) as follows in 'Running the app' section. 



This app uses port 3000, so make sure this port is available in your pc so the app can run succesfully.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
